Enhanced GitHub

Browser Extension that provides GitHub add-ons

Features

File Size

Isn't it useful to know the file-size before attempting to view it? With this Extension, View size of every file listed in the file-navigation view.

Download File

Ever wonder to download just a file instead of cloning/downloading the entire reposiroty? Download a particular file on your system with a single click using this extension.

Copy File Contents

Do you copy file contents from GitHub often? With just one click, directly copy file contents to the clipboard.

Raw File Link

Get File's raw link directly. GitHub raw links help in lot ways. Use it to view markdown(and such) files raw-code.

Repo Size

Curious to know how big is the repo before cloning/downloading it? This Extension dissplay the size of the entire repo on the repo homepage.

Fully Open Sourced

The entire extension is open-sourced on GitHub and licensed under MIT. Feel free to contribute or report any bug/suggestion.

Privacy Policy



Enhanced GitHub Chrome Extension doesn't collect/share/care about your data at all.
Asks for GitHub Access Token only for private repos and if Rate Limiting is your concern.
Permissions Required:
  • "*://*.github.com/*" - for running on github.com domain.
  • storage - for storing GitHub access token on your local system. GitHub Access token is required for private repos and for handling API rate-limiting issues.
  • webRequest - to support GitHub being a single-page-application now.
  • webNavigation - to support GitHub being a single-page-application now.



Read the detailed blog which explains why above-mentioned permissions are required to support GitHub SPA website.
